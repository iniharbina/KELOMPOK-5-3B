Prediksi Angka Kemiskinan di Indonesia

Aplikasi ini adalah sebuah aplikasi yang digunakan untuk memprediksi angka Kemiskinan di Negara Indonesia
Dengan menggunakan Machine Learning, Modelling Supervised Learning dan Motode Decission Tree yang mengimplementasikan Library SKLEARN dan mengambil fungsi DecisionTreeRegressor
Aplikasi ini akan memprediksi angka kemiskinan di Negara Indonesia berdasarkan data yang diperoleh
dengan memasukkan Regional GDP, Life Expectancy, Average Schooltime, dan Expenses per Capita machine learning akan
memproses lalu memberikan prediksi angka kemiskinan menurut parameter yang telah diinputkan

Supervised learning adalah salah satu metode pembelajaran mesin yang paling umum digunakan. Metode ini menggunakan 
data yang sudah diberi label untuk melatih model pembelajaran mesin. Label pada data tersebut mewakili output yang 
diinginkan dari model.

Pada supervised learning, model pembelajaran mesin belajar untuk memprediksi output dari input yang baru. 
Misalnya, model pembelajaran mesin dapat dilatih untuk memprediksi harga rumah berdasarkan data tentang lokasi, 
ukuran, dan fitur rumah tersebut.

Supervised learning dapat digunakan untuk berbagai macam tugas, termasuk:

1. Klasifikasi: Mengklasifikasikan data ke dalam kategori yang berbeda. Misalnya, model pembelajaran mesin dapat
   digunakan untuk mengklasifikasikan email sebagai spam atau bukan spam.
2. Regresi: Memprediksi nilai kontinu. Misalnya, model pembelajaran mesin dapat digunakan untuk memprediksi harga 
   saham pada hari berikutnya.
3. Pengaturan: Menghasilkan output yang diinginkan. Misalnya, model pembelajaran mesin dapat digunakan untuk menghasilkan 
   teks yang mirip dengan teks yang sudah ada.

Supervised learning berbeda dengan unsupervised learning. Pada unsupervised learning, model pembelajaran mesin belajar dari data yang tidak diberi label.

Decision Tree

Decision tree adalah algoritma supervised learning yang dapat digunakan untuk masalah klasifikasi dan regresi. Pada masalah klasifikasi, decision tree digunakan
untuk mengklasifikasikan data ke dalam kategori yang berbeda. Misalnya, decision tree dapat digunakan untuk mengklasifikasikan email sebagai spam atau bukan spam. 
Pada masalah regresi, decision tree digunakan untuk memprediksi nilai kontinu. Misalnya, decision tree dapat digunakan untuk memprediksi harga rumah berdasarkan
data tentang lokasi, ukuran, dan fitur rumah tersebut.

Decision tree bekerja dengan membangun struktur pohon yang terdiri dari node dan cabang. Node akar mewakili data awal, dan node daun mewakili kategori atau nilai yang diprediksi. 
Cabang mewakili pertanyaan atau aturan yang digunakan untuk membuat keputusan tentang bagaimana data akan diklasifikasikan atau diprediksi.

Pada saat pelatihan, decision tree menggunakan data yang sudah diberi label untuk mempelajari aturan atau pertanyaan yang terbaik untuk memprediksi output yang diinginkan. Setelah dilatih, 
decision tree dapat digunakan untuk memprediksi output dari input yang baru.
